   SEARCH,PRODUCTION = SEARCH,production = SEARCH,ProDuCtion
   
2. Use dates to find old sessions
   SEARCH,20251106 = Find files from Nov 6, 2025
   
3. Use times to find specific time range
   SEARCH,14 = Find files around 2 PM
   
4. Filenames starting with "/" are absolute paths
   READ,/Production_20251107_143025_to_143530.txt
   READ,Production_20251107_143025_to_143530.txt
   (Both work the same)
   
5. Always check LS before READ to verify filename
   
6. Use PROD command to quickly find production files
   (Faster than searching)

  1. Use LS to see actual filenames
  2. Check filename spelling (case-sensitive)
  3. File might have been deleted

Solution:
  1. Check baud rate is 115200
  2. Check USB cable connection
  3. Try pressing ESP32 RESET button
  4. Select correct COM port

  1. Check command spelling
  2. Make sure comma is included (e.g., SEARCH,text)
  3. Type INFO to see valid commands
